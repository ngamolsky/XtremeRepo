# Lake Tahoe Relay raw results files

Source: https://laketahoerelay.com/results/

This directory contains the raw linked result files from the source page, plus `manifest.json` and `manifest.csv` with source URLs, file types, byte sizes, and SHA-256 checksums.

Notes:
- 2024 and 2025 are only linked as PDFs on the source page.
- 2000-2003 are linked as image scans.
- The remaining linked years are Excel workbooks.
